Grid Flow

Grid layout with the display property

Specify columns using the grid-template-columns property:

.grid-parent {
    display: grid;
    /* grid-template-columns: 25% 75%; */
    grid-template-columns: 1fr 3fr;
    gap: 1em;
    margin-bottom: 1em;
}

.grid-child {
    background-color: aqua;
    padding: 0.5em;
}
